# 5-Channel-AC-Dimmer-and-Switch-Module-with-STM32
A high-voltage system capable of switching and smoothly dimming up to five independent 220V AC channels. Driven by an onboard STM32F103C8T6 microcontroller, the module acts as a versatile hardware interface that can operate completely independently with physical inputs or function as a modular peripheral controlled by an external master system.
